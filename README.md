# WallaHop Privacy Policy – Hosting Instructions

## Option 1: GitHub Pages (Free)

1. Create a new repo on GitHub (e.g. `wallahop-privacy`)
2. Upload the `index.html` file to the root of the repo
3. Go to repo **Settings** → **Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** (or **master**), folder: **/ (root)**
6. Save – your URL will be: `https://YOUR_USERNAME.github.io/wallahop-privacy/`

**Privacy Policy URL for App Store Connect:** `https://YOUR_USERNAME.github.io/wallahop-privacy/`

---

## Option 2: Netlify (Free)

1. Go to [netlify.com](https://www.netlify.com) and sign up (free)
2. **Add new site** → **Deploy manually**
3. Drag the `privacy-policy` folder (containing index.html) into the drop zone
4. Netlify gives you a URL like: `https://random-name-12345.netlify.app`
5. Optional: Add a custom domain or change the subdomain in site settings

---

## Option 3: Cloudflare Pages (Free)

1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. **Create a project** → **Direct Upload**
3. Upload the `privacy-policy` folder
4. You get a URL like: `https://wallahop-privacy.pages.dev`

---

## Recommended

**GitHub Pages** is usually fastest if you already have GitHub – no extra signup, and the URL looks like: `https://yourusername.github.io/wallahop-privacy/`
